SV-Health Base Registries [Motivation] ()
NARRATIVE: The SV-Health Base Registries [Motivation] view extends the Semantic view and provides a comprehensive framework for leveraging various health base registries. These registries contain essential data related to healthcare services, treatments, medical equipment, patient information, and more. The view includes three main elements: Health Base Registry Catalogue, Health Business Information Base, and Health Base Registry Solutions. 1. Health Base Registry Catalogue: - The Health Base Registry Catalogue is a grouping element that represents a collection of various base registries. - It includes several data objects, each representing a specific base registry: - Drugs interactions registry: A base registry containing all the available drugs interactions. - Authorised treatments registry: A base registry containing the list of all the possible legally authorised treatments. - Authorised medical images registry: A base registry containing the list of all the types of medical images that can be legally prescribed/taken. - Authorised medical equipments registry: A base registry containing the list of all the legally authorised medical equipments. - Authorised medical devices registry: A base registry containing a list of all the legally authorised medical devices. - Imaging bank for test and experimentation facilities: A data set containing all the medical images to be used for testing and experimental purposes. - Credentials registry: A base registry containing all the credential data to access public administration information systems. - Health professional qualifications registry: A base registry containing the list of all the qualifications that can be associated with healthcare professionals. - Diagnosis registry: A base registry containing all the diagnoses performed by authorised health professionals. - ADT (Admission Discharge and Transfer) registry: A base registry containing all the data about patients' admissions, discharges, and transfers. - Authorised vaccines registry: A base registry including all the types of vaccines and vaccination treatments that can be legally dispensed. - Eligible patients registry: A data repository that holds and organizes data about eligible patients for specific research domains or subjects. - Hospital tickets registry: A base registry containing the data about the daily processed patient hospitalisation and consultation tickets. - Prescriptions registry: A base registry containing all the data about the prescriptions issued by health professionals or healthcare facilities to patients. - Laboratory analyser data registry: A base registry containing all the data about the different chemicals and other characteristics in biological samples analyzed by the laboratory analyser instrument. - Laboratory pre/post-analytical device data registry: A base registry containing all the data exchanged between an Automation Manager and a set of automated Laboratory Devices to process work orders and retrieve results. - Treatments registry: A base registry containing all the data about the treatments prescribed to inpatients or outpatients. - Vaccinations registry: A base registry including all the data about the vaccinations performed by the national health system. - Patients registry: A base registry containing all the patient administrative data, such as demographic data, hospitalization data, localization data, etc. - Healthcare payments registry: A base registry containing all the data about the payments performed by patients to access healthcare services. - Healthcare facilities registry: A base registry containing all the data about healthcare facilities. - Dispensation reports registry: A base registry containing all the data about the medicines to be dispensed and/or that have been dispensed. - Laboratory reports registry: A base registry containing all the data about the laboratory reports performed by a patient. - Medical images registry: A base registry containing all the data about the medical images captured in hospitals, clinics, and other legally entitled sites. - Patient summary registry: A base registry containing all the national patient summaries of patients. - Healthcare bills registry: A base registry containing all the data about the bills related to healthcare services. 2. Health Business Information Base: - The Health Business Information Base is a grouping element that represents a collection of various business objects related to health information. - It includes several business objects, each representing a specific type of health information: - Medical product: Contains information about the summary of product characteristics, contained substances, package leaflets, and labeling. - Medical equipment: Contains information about the summary and specifications of the available and authorized medical equipment. - Medical imaging: Contains information about medical imaging diagnosis, including X-rays, magnetic resonance imaging (MRI), ultrasounds, etc. - Hospital Enterprise Resource Planning: Contains information about the management of hospital resources for the execution of health processes, such as surgery scheduling, beds availability, consultation scheduling, etc. - Diagnosis: Contains information about the details related to the patient's diagnosis, such as symptoms and anamnesis. - Patient clinical: Contains information about patient clinical data, including medical records and reports kept in the ordinary course of a physician's business. - Treatment: Contains information about the treatments legally authorized by the defined health authority that can be prescribed to a patient. - Vaccination: Contains information about the administered vaccines and their effects on the health of patients. - Patient eligibility: Contains information about the eligibility of patients to receive specific treatments, such as clinical trials. - Hospital administrative: Contains information about hospital-related aspects, such as financial and demographic information. - Prescription: Contains information about a prescription, including the plan of care for an individual patient. - Laboratory: Contains information about the laboratory exams performed by a patient in a healthcare facility. 3. Health Base Registry Solutions: - The Health Base Registry Solutions is a grouping element that represents a collection of specific solutions related to health base registries. - It includes a data object representing a specific solution: - European Reference Laboratory Network for TB (ERLTB-Net): A network launched by ECDC to support countries in ensuring reliable and timely diagnostic services for tuberculosis outbreaks. These elements are organized hierarchically, with the Health Base Registry Catalogue and Health Business Information Base being the top-level groupings, and the specific base registries and business objects being the sub-elements within them. The motivation behind this view is to provide a comprehensive framework for leveraging the data and information contained in these registries to support various healthcare processes and decision-making.
